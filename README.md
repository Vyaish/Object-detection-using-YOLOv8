# 👥 People Detection Web App (YOLOv8)

## 🚀 Overview

This project is a computer vision application that detects and counts people in images using YOLOv8.

## ✨ Features

* Upload multiple images
* Detect objects using YOLOv8
* Count number of people in each image
* Display annotated images
* Simple UI using Streamlit

## 🛠 Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```


## 📌 Use Cases

* Crowd monitoring
* Smart surveillance
* Retail analytics
